# Teralya

Marketplace europeo de bodegas fundadoras.

## Estructura del repositorio

```
teralya/
├── docs/
│   ├── CAP/             Documentación funcional (Casos de Uso, Pantallas)
│   ├── INF/              Documentación técnica (arquitectura, esquema, APIs)
│   ├── ADR/               Registros de decisiones de arquitectura
│   ├── DECISION_LOG/       Registro de decisiones de negocio y producto
│   ├── LEGAL/               Documentación legal y de cumplimiento
│   └── UX/                   Diseño de experiencia de usuario
├── frontend/            Aplicaciones web (Comprador, Panel de Bodega)
├── backend/             API y lógica de negocio
├── database/            Esquema y migraciones
├── infrastructure/      IaC y CI/CD
└── assets/              Recursos de marca
```

Documentación oficial del proyecto, subida progresivamente a `docs/`.
